This here web site is not the coolest site - and it might not be perfectly constructed. However, it is my intention to give you an idea of how to start structuring a basic html page. Strap in.

So, html content is basically a set of boxes aligned next to each other - or on top of each other. The display property of html elements can be set (amongst other values) to block, inline or inline-block. There are a lot of other values we can use, but let's settle for these - for the time being.

First we need to understand that the block elements typically start a new line and also reserve the whole width of their container. So, you might say that they 'block' other elements. Block-level elements are e.g. p, div, h1-h6, hr. You can find the complete list here.

Generally block-level elements can contain inline elements and (sometimes) other block elements. Now, on your html page (this page is 'index.html') you specify the content visible on this screen. Try to refrain from manipulating the styling of your page within your html code. Instead use a CSS file for that.

Up until now we have used CSS classes and id properties. All of these definitions and rules origin from a single file called 'styles.css'. Actually we have used LESS which is basically CSS, with just a little more. But we have time to talk about that later. My point was that so far everything has been handwritten. Still, that needs not to be the case. Amongst other cool things which are nowadays available for us developers there is for Bootstrap.

This is to say, that we actually don't need to do all the styling from the scratch, and even while using Bootstrap, you can always further modify these ready-made bootstrap classes. Because, at the end of the day, bootstrap is just a one gigantic CSS file.

So, that's it. Just a simple single page implementation. Use (preferably) a Chrome browser and check out how this page is constructed. Use Chrome developer tools and you can even add an extension called 'Pesticide'. And of course you can visit the repository of this project at GitHub.

At the HTML-CSS part of your course we will go over subjects like HTML box-model, utilizing Emmet abreviations, CSS-LESS, Visual Studio Code and just a little bit of AngularJS directives. See you then!
